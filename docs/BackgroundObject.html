Class: BackgroundObject

BackgroundObject(imagePath, x)

Repräsentiert ein Hintergrundobjekt (z. B. Berge, Bäume, Himmel), das sich im Level verschiebt. Erbt Bewegung und Zeichenfunktionen von MoveableObject.

Constructor

new BackgroundObject(imagePath, x)

Erstellt ein Hintergrundobjekt mit gegebenem Bild und X-Position.
Parameters:
Name Type Description

x number X-Position, an der das Objekt erscheinen soll
